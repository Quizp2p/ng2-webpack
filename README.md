# Angular2 Frontend Architecture Design

## Details

### package manager

npm

### css pre processor

less

### module manager and build tool

webpack

### unit test

karma

## Useage

### download

```bash
git clone https://github.com/yuyang041060120/ng2-webpack.git
```

### install

```bash
npm install
```

### development mode

```bash
npm start

http://localhost:3200
```

### production mode

```bash
npm run build
```
